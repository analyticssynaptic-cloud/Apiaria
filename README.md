# ApiarIA Studio · HTML Forge

Colmena creativa donde los agentes **debaten entre sí** antes de construir: Propuestas → Debate cruzado → Plan de consenso → Construcción → Auditoría → Refinado. La pieza resultante **se ejecuta en vivo** en un canvas integrado (vista previa, móvil, código, descarga).

**Características:** 10 agentes creativos (Arquitecto, Director de Arte, Animador, Interacción, Responsive, Accesibilidad, Copywriter, UX, Performance, Datos) · Skills (sistemas de diseño propios que la colmena cumple obligatoriamente) · Memoria de proyectos · Contexto desde repos públicos de GitHub · Multi-proveedor (Gemini gratis / Claude / Perplexity) · Modo edición quirúrgica sobre piezas existentes.

**Instalación:** sube `index.html`, `hive.js` y `providers.js` a un repo público con GitHub Pages. Detalles en `INSTRUCCIONES-STUDIO.txt`.
